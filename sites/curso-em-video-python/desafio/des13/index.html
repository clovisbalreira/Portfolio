Desafio 13 - Faça um programa que leia a largura e a altura de uma perede em metros, calcule a sua área e a quantidade de tinta necessária para pintá-la, sabendo que cada litro de tinta pinta uma área de 2m².

def dados(*arg, **kags): input_altura = Element('altura') altura = input_altura.element.value input_largura = Element('largura') largura = input_largura.element.value resultado = Element('resultado') resultado.element.innerText = 'A parede tem {:.2f} de altura e {:.2f} de largura, e sua área quadrada é {:.2f}m² \n você precisa de {:.2f}l para pintar a parede'.format(float(altura),float(largura),float(altura)*float(largura),float(altura)*float(largura) / 2)
